# 2026-03-24 — BourbonFoxhound MVP Complete

## What We Built Today

Full-stack bourbon tracking platform — "Untappd for bourbon" with education layer.

### Core Features
- **Auth**: Login/signup with Supabase
- **Add Review**: Photo upload, 5-star rating, Mapbox location picker, notes
- **Map View**: All reviews as pins, list view toggle
- **Feed**: Agent-curated drops/news from automated web scraping
- **Learn Section**: Bourbon 101 education with free/premium gating

### Tech Stack
- Next.js 15 + TypeScript + Tailwind CSS
- Supabase (Auth, Database, Storage)
- Mapbox GL (location tagging)
- GitHub Pages (agent JSON feed)
- Cron jobs: Agent scrapes 2x daily, syncs to GitHub

### Education Content (3 Free, 3 Premium)
- What is Bourbon? (free)
- How to Taste Bourbon — The 5 S's (free)
- Understanding Mashbills (free — to build)
- Major Distilleries (premium)
- Reading Labels (premium)
- Collecting vs Drinking (premium)

### Subscription Model
- **Free**: 3 lessons, basic reviews, public feed
- **Premium ($4.99/mo)**: All lessons, price alerts, early drop news, advanced notes

"First location-tagged bourbon tracking platform. Users post reviews with GPS. Agents scan web for releases. We own the data layer connecting consumption to location — something no whiskey app has cracked."
