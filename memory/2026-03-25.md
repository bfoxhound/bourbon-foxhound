# Memory Log — 2026-03-25

## 🥃 BourbonFoxhound Project

**Concept:** Social platform for bourbon enthusiasts — "Untappd for bourbon, but the data is actually valuable because scarcity drives price."

### Key Architecture Decisions
- **Frontend:** Next.js PWA (cross-platform, no app store needed)
- **Backend:** Supabase (Postgres, open source, no vendor lock-in)
- **Maps:** Mapbox (cheaper than Google, 50k free loads)
- **AI Development:** Kimi (Moonshot AI) via OpenClaw — NOT self-hosted Claude/Anthropic
- **Hosting:** Vercel (edge-deployed, serverless)
- **Source Control:** Dedicated GitHub repo `bourbonfoxhound-app`

### Business Model Insights
- **Primary revenue:** Data licensing to distilleries (they pay for consumption geography)
- **Secondary:** Affiliate commerce (Drizly/ReserveBar 5-15% on referred sales)
- **Key differentiator:** GPS-tagged consumption data showing *where* bourbon is drunk vs. hoarded
- **Break-even:** ~400 premium subscribers OR ~2,000 active users with affiliate mix

### MVP Scope
- Landing page with embedded agent feed
- Auth (email/password + social)
- Add Review (photo, rating, GPS location)
- Map view with review pins
- Learn section (3 free lessons, 3 premium)

### Wix Interim Strategy
- Using existing Wix site as landing page
- Embedded GitHub Pages feed shows live bourbon intelligence
- Bridge solution while full app is built
- Agent feed repo: `bourbon-foxhound-data`

### Agent Automation Setup
- **bourbon-monitor** agent scrapes 2x daily (8 AM, 8 PM UTC)
- Sources: Buffalo Trace, Reddit r/bourbon, Breaking Bourbon RSS
- GitHub Actions runs the scraper, commits JSON data
- HTML feed auto-refreshes every 5 minutes

### Investor Narrative
- Problem: $9B bourbon market growing 8% YoY, collectors spend $10k on bottles but track in Excel
- Solution: Location-tagged reviews + automated intelligence on drops/prices
- Moat: 6 months of location-tagged consumption data (competitors can copy app, not dataset)
- Traction targets: X users → Y reviews → Z locations → 1 affiliate partnership

### Operating Costs
- MVP (0-1k users): $20/mo (Vercel Pro)
- Growth (1k-10k users): $70-110/mo
- Scale (10k+ users): $550-1,300/mo

### 30-Day Demo Roadmap
- Week 1: Working prototype (Auth + Add Review + Map)
- Week 2: Data pipeline demo (agent logs + trending)
- Week 3: Polish + onboarding (<60 sec first review)
- Week 4: Pitch assets (demo video + architecture diagram)
